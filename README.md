# Decentralized data labeling (Solana)

Portfolio project: off-chain labeling + on-chain verification via an oracle. Phases: 1 Anchor program, 2 backend + SQLite, 3 oracle script, 4 React frontend, 5 polish + optional multimodal.

## Phase 1 – Anchor program (this repo)

- **Workspace:** [anchor_data/](anchor_data/) – Rust/Anchor program only. Run all `anchor` commands from inside **anchor_data/** (e.g. `cd anchor_data` then `anchor build`).
- Build and deploy: see [anchor_data/README.md](anchor_data/README.md).
- Next: Phase 2 will add a Node backend and SQLite under a separate folder.
